# Changelog

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-04-22

### Added
- Initial release of Project Arjun (God's Eye India)
- CesiumJS 3D globe with Maharashtra focus
- Maharashtra district boundaries with GeoJSON data
- Water quality monitoring layer (CPCB data)
- Air quality monitoring layer (CPCB stations)
- Health infrastructure layer (MoHFW data)
- Agriculture layer (agriculture department data)
- Infrastructure layer (PWD, Metro, NHAI projects)
- Government schemes layer with scheme details
- Rainfall & weather layer (IMD data)
- News & events layer (GDELT integration)
- Satellite coverage tracking layer
- Dashboard component with layer toggle controls
- Timeline navigation component
- DataStore with fetch & caching support
- GitHub Actions CI/CD for GitHub Pages deployment
- Vite build system with CesiumJS plugin
- TypeScript codebase with full type safety
- MIT license

### Technical
- Vite 6.3 build tooling
- CesiumJS 1.127 globe engine
- TypeScript 5.8
- GitHub Pages deployment via GitHub Actions
- Zero-cost operation with open data sources
